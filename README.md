📄 AI Resume Scanner
An AI-powered project that automatically scans, analyzes, and ranks resumes. It helps in speeding up the recruitment process by extracting important information like skills, education, and experience.

🚀 Features
📄 Resume parsing (PDF/DOC)
🤖 AI-based analysis
🔍 Skill extraction
📊 Candidate scoring & ranking
⚡ Fast processing
📁 Structured output
🧠 How It Works
Upload resume
Extract text from file
Analyze using NLP
Match with job requirements
Generate score and output
🛠️ Tech Stack
Python
NLP (Natural Language Processing)
Flask / Django
PyPDF2 / pdfplumber / docx
MongoDB / MySQL
